   $npx tailwindcss init -p
- reemplazo el contenido de taIlwind.config.js por este:
/** @type {import('tailwindcss').Config} */
export default {
  content: [
    "./index.html",
    "./src/**/*.{js,ts,jsx,tsx}",
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}

-Reemplazo el contenido de src/index.css por este:
